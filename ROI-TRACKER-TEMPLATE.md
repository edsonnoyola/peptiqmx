## FÓRMULAS ÚTILES PARA GOOGLE SHEETS

**ROAS (Return on Ad Spend):**
```
=Revenue/Ad_Spend
```
Ejemplo: Si gastaste $500 y vendiste $3,000 → ROAS = 6

**CAC (Customer Acquisition Cost):**
```
=Ad_Spend/Ventas
```
Ejemplo: Si gastaste $3,500 y tuviste 15 ventas → CAC = $233

**Conversion Rate (WhatsApp):**
```
=(Ventas/WhatsApp_Msgs)*100
```
Ejemplo: Si recibiste 30 mensajes y cerraste 5 ventas → CR = 16.7%

**AOV (Average Order Value):**
```
=Revenue/Ventas
```
Ejemplo: Si vendiste $50,000 en 20 ventas → AOV = $2,500

**Profit Neto:**
```
=Revenue - COGS - Ad_Spend - Costos_Fijos
```
