## 📋 Setup TikTok Ads en orden

1. **Cuenta:** https://ads.tiktok.com/i18n/signup/ · datos negocio MX (RFC + dirección)
2. **Pixel:** Assets → Events → Web Events → Create Pixel → copy ID
3. **Pasarme el Pixel ID** → yo lo activo en `/evaluacion`
4. **Subir 5 videos** como posts orgánicos primero en @peptiqmx_mx
5. **Spark Ads** para promover los que tengan mejor engagement orgánico
6. **Audiencia inicial:** MX 28-58 · interés "wellness · longevity · biohacking · skincare · health"
7. **Lookalike** después de 50 conversiones: similar a quiz completers
