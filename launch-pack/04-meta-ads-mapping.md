## 🚨 Reglas Meta · checklist por ad antes de subir

| ✓ | Regla |
|---|---|
| ☐ | NO menciona péptido por nombre en imagen (BPC-157, Tesamorelin) en headline grande |
| ☐ | Microtext con ingredientes está OK pero solo "Research Grade" si es ad principal |
| ☐ | Brand line "PEPTIQ MX · Research Use Only" visible en imagen |
| ☐ | NO dice "antes / después" |
| ☐ | NO promete números (kg, años, %) |
| ☐ | NO compara con Ozempic / Rx |
| ☐ | NO shirtless / NO sports bra exposed |
| ☐ | Texto en imagen <20% del área |
| ☐ | Disclaimer "Material para investigación de laboratorio" en footer |
| ☐ | CTA button: "Más información" o "Aprender más" (NUNCA "Comprar") |
